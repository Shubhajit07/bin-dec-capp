# Decimal to Binary or Binary To Decimal converter

This is a simple python based console app to convert Binary value to Decimal or Decimal value to Binary

## Install required libraries

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install libraries.

```bash

pip install colorama

```

```bash

pip install termcolor

```
